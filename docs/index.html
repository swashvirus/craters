Getting started
it includes all basics you need to get a game up and running in a few lines of code leaving enough room for the game logic. suitable for js13kgames competition
  • lightweight and portable so minimal that it's just 1.5kb in size! zipped and minified.

Game method
game is a method which creates an instance of a game world
  • the game and entities both have methods update and render.
  • the games instance can have entities and entities can also have their own entities.
  • Entities inherit the craters.entity class.
